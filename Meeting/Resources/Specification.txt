Статусы объектов модели:

	1) Чат. Сущность "Chat"
		Статусы:
			1ХХХ - Маска чата типа "совещание".
			10ХХ - По умолчанию. Активное совещание. 
			11ХХ - Совещание прервано. Пользователи временно не могут писать сообщения.
			12ХХ - Совещание завершено и не может быть продолжено.
			1ХХ0 - Протокол отсутствует. (Готовый протокол нигде не хранится, создается по требованию. Наличие протокола - индикатор того, что админиистратор чата создавал протокол и отправлял его на утверждение.)
			1ХХ1 - Наличие протокола, не подтвержденного хотя бы одним учатником совещания. (Данный протокол должен быть помечен знаком "Не утвержденный проект протокола" или что-то такое)
			1ХХ2 - Наличие протокола, подтвержденного КАЖДЫМ участником совещания. (При наличии подтвержденного протокола совещание должно быть завершено).

	2) Пользователь. Сущность "User" 
		Статусы:
			1ХХХ - Участник чата.
			2ХХХ - Администратор.
			X0ХХ - Не посещал чат ни разу.
			Х1ХХ - Посещал чат.	
			Х2ХХ - Оставлял сообщения в чате (для подтверждения протокола только пользователями, которые отправляли сообщение.)
			Для типа чата "Совещание":
			ХХХ0 - Не голосовал.
			ХХХ1 - Подтвердил протокол.
			ХХХ2 - Отклонил протокол.

	3) Сообщение. Сущность "Message"
		Стутусы:
			1000 - По умолчанию.
			Для типа чата "Совещание":
			ХХХ0 - Не включено в протокол.
			ХХХ1 - Включено в протокол.
